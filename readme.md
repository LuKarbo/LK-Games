
# LK-Games (PHP) [Test 1]
LK-Games es una web creada como proyecto para la materia Web2, con el objetivo de mejorar el uso de PHP e integraciones.
Ademas de aplicar patrones de diseño y estructurar de forma correcta el contenido del proyecto

### Funcionalidades Principales
- Registro y Login de usuarios.
- Filtro de listado.
- Crud de Juegos.
- Crud de Usuarios.

### Tecnologías Utilizadas
- PHP para la lógica.
- SQL para la creación de Consultas.

### Estructura del Proyecto
El proyecto está estructurado en las siguientes partes:
- `app/`: Contiene todo el contenido lógico/modelos/accesos
- `public/`: Contiene el apartado público de la web, sus distintas paginas y controladores.

#### Nota
- El guardado de imagenes esta hecho de forma erronea, ya que se esta guardando en la BBDD cuando debería de hacerce de forma local para evitar sobrecargar de contenido la BBDD.
